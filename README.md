# 🚀 DevOps CI/CD Pipeline Project

## 📌 Project Overview

This project demonstrates a complete CI/CD pipeline using **GitHub Actions and Docker** to automate the build and execution of a Node.js application.

The goal of this project is to simulate how modern DevOps workflows automate software delivery in real-world environments.

---

## ⚙️ What I Have Implemented

* ✅ Automated CI/CD pipeline using GitHub Actions
* ✅ Triggered workflow on code push to `main` branch
* ✅ Dockerized the application using a Dockerfile
* ✅ Built Docker image automatically in pipeline
* ✅ Ran container to verify application execution
* ✅ Structured workflow using YAML configuration

---

## 🔁 How It Works

1. Developer pushes code to GitHub (`main` branch)
2. GitHub Actions automatically triggers the workflow
3. Workflow performs:

   * Checkout source code
   * Build Docker image
   * Run Docker container
4. Application runs successfully inside container
